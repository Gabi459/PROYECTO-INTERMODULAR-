# AgendaPro - Proyecto Web

## ¿Qué empresa representa la web?

AgendaPro es una plataforma digital que simula una empresa tecnológica dedicada al desarrollo de herramientas de organización personal. Su objetivo principal es ayudar a los usuarios a gestionar su tiempo de forma eficiente mediante la planificación de tareas, eventos y recordatorios en un entorno sencillo e intuitivo.

Este proyecto ha sido desarrollado como parte de un trabajo intermodular educativo, integrando conocimientos de desarrollo web, diseño y programación.

---

## ¿Qué páginas incluye?

La web de AgendaPro está compuesta por las siguientes páginas:

* **Inicio (index.html)**
  Página principal donde se presenta la aplicación y sus funcionalidades principales.

* **Calendario (calendario.html)**
  Vista tipo agenda donde se organizan los días y eventos.

* **Tareas (tareas.html)**
  Sección para visualizar y gestionar tareas pendientes y completadas.

* **Sobre nosotros (sobre.html)**
  Información sobre el proyecto y sus desarrolladores.

* **Contacto (contacto.html)**
  Formulario o información de contacto.

* **Login (login.html)**
  Pagina en la que se introduce el usuario y al contraseña

---

## ¿Cómo abrir o visualizar la web?

Para visualizar correctamente el proyecto:

1. Descarga o copia la carpeta completa del proyecto.
2. Asegúrate de que mantiene esta estructura:

   * `/index.html`
   * `/styles/`
3. Haz doble clic en el archivo **index.html**
   o ábrelo con cualquier navegador web (Chrome, Edge, Firefox, etc.).

Es importante no mover los archivos de su estructura original para que los estilos (CSS) se carguen correctamente.
